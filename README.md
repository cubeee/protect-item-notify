# Protect Item Notify Plugin
This plugin allows you to have the protect item icon displayed when the protect item prayer is disabled.

Enabled in:
* Normal worlds: wilderness
* PvP Worlds: outside safe zones
* Bounty Hunter: outside safe zone

Disabled in:
* Fight pits
* Last Man Standing
* PvP Worlds: safe zones
* Clan Wars
* Castle Wars
* Soul Wars
* Emir's Arena
